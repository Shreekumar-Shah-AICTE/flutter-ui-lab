### 📚 Academic Title: **CRUD Operations using SQLite Database**

> Mobile Programming Practical #11 — Prof. Akash Padhiyar

A color-coded note-taking app with full Create, Read, Update, Delete operations using SQLite.

**Key Features:**
- Full CRUD via `sqflite` package
- Color-coded notes with 6 pastel options
- Masonry-style grid layout
- Search filtering across title and content
- Long-press to delete with confirmation dialog
- Note editor with dynamic background color
- Timestamps on each card (sorted by most recent)

**State Management:** `setState` + `sqflite` + `DatabaseHelper` singleton
