# 06 · Catalog
### 📚 Academic Title: **ListView Implementation**

> Mobile Programming Practical #6 — Prof. Akash Padhiyar

A magazine-style staggered content feed with shimmer loading, pull-to-refresh, and swipe-to-dismiss.

**Key Features:**
- Staggered entrance animations (each card delayed by 60ms)
- Shimmer loading skeleton during simulated data fetch
- Pull-to-refresh with custom indicator
- Swipe-to-dismiss with red delete background
- Category pills with color-coded badges
- Article count tracker

**State Management:** `setState`
