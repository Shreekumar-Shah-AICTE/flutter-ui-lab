# 10 · Dispatch
### 📚 Academic Title: **News Application (Using API)**

> Mobile Programming Practical #10 — Prof. Akash Padhiyar

A premium news reader powered by GNews API with category tabs and article detail pages.

**Key Features:**
- GNews API integration (no auth required with demo key)
- Horizontal scrolling category chips with animated active state
- Article cards with thumbnail images
- Shimmer loading skeleton during API fetch
- Article detail page with hero image
- Pull-to-refresh and error state with retry
- Source label in monospace accent color

**State Management:** `setState` + `http` package
