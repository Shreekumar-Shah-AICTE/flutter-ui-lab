# 09 · Storefront
### 📚 Academic Title: **E-Commerce App UI (Using API)**

> Mobile Programming Practical #9 — Prof. Akash Padhiyar

A premium e-commerce storefront powered by FakeStore API with product grid, cart, and detail pages.

**Key Features:**
- REST API integration with FakeStore API
- Product grid with staggered entrance animations
- Shimmer loading skeleton during API fetch
- Add-to-cart toggle with badge counter
- Product detail page with full description
- Star rating display
- Pull-to-refresh, network error state with retry

**State Management:** `setState` + `http` package
