# 05 · Enroll
### 📚 Academic Title: **Student Registration Form with Validation**

> Mobile Programming Practical #5 — Prof. Akash Padhiyar

A 3-step onboarding wizard with real-time validation, animated progress bar, and a satisfying success animation.

**Key Features:**
- 3-step wizard: Personal Info → Academic Details → Review
- Animated progress bar with step labels
- Real-time field validation with error messages
- Dropdown for Branch selection, stepper for Semester
- Review card showing all entered data before submission
- Elastic scale success animation on completion

**State Management:** `setState` + `GlobalKey<FormState>`
