# 01 · Pulse
### 📚 Academic Title: **Counter Application**

> Mobile Programming Practical #1 — Prof. Akash Padhiyar

A neumorphic haptic-feel counter with fluid spring animations, circular progress ring, and satisfying tactile interactions.

**Key Features:**
- Neumorphic circular dial with dual shadows
- Spring scale animation on increment/decrement
- Circular progress ring (resets at 100)
- AnimatedSwitcher for smooth number transitions
- Haptic feedback on every interaction
- Long-press to reset with elastic bounce-back

**State Management:** `setState`
