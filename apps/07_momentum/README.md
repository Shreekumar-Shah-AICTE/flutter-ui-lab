# 07 · Momentum
### 📚 Academic Title: **To-Do Application**

> Mobile Programming Practical #7 — Prof. Akash Padhiyar

A gesture-driven task manager with satisfying completion animations and swipe-to-delete.

**Key Features:**
- Add tasks via bottom sheet with text input
- Tap to toggle completion with animated checkbox
- Strikethrough text animation on completion
- Swipe left to delete with red background reveal
- Completed tasks sorted below a divider
- Empty state with friendly message
- Haptic feedback on all interactions

**State Management:** `setState` + `List<Task>`
