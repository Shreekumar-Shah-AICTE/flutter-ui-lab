# 02 · Gateway
### 📚 Academic Title: **Simple Login UI**

> Mobile Programming Practical #2 — Prof. Akash Padhiyar

A cinematic, glassmorphic authentication experience with frosted glass card, ambient gradient orbs, and biometric hint animation.

**Key Features:**
- Full-bleed dark gradient background with ambient orbs
- BackdropFilter frosted glass card
- Animated floating labels on input fields
- Password visibility toggle
- Loading spinner state on sign in
- Pulsing biometric fingerprint icon

**State Management:** `setState` + `TextEditingController`
