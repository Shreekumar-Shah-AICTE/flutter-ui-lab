# 03 · Calc.Noir
### 📚 Academic Title: **Simple Calculator**

> Mobile Programming Practical #3 — Prof. Akash Padhiyar

A dark, Teenage Engineering-inspired hardware synth calculator with monospace display, text glow, and micro-interaction press animations.

**Key Features:**
- Pure black aesthetic with JetBrains Mono typography
- Soft text glow on display via shadow
- Micro scale-down (0.92) button press animation
- Full arithmetic: +, −, ×, ÷, %, ±, backspace
- Division by zero shows: "∞ — nice try."
- Expression preview above main display

**State Management:** `setState`
