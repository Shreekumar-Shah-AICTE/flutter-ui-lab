# 04 · Atlas
### 📚 Academic Title: **Navigation and App Icon**

> Mobile Programming Practical #4 — Prof. Akash Padhiyar

A multi-page navigation showcase with fluid Hero transitions, morphing bottom navigation bar, and gradient content cards.

**Key Features:**
- 4 pages: Home, Explore, Profile, Settings
- Animated bottom nav bar with sliding pill indicator
- Hero shared-element transitions between Home cards and Detail page
- Gradient cards with shadow tinting
- Category grid on Explore page
- Profile stats with monospace values

**State Management:** `setState` + `Navigator`
