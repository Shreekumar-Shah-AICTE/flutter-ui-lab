# 08 · Vault
### 📚 Academic Title: **SharedPreferences Implementation**

> Mobile Programming Practical #8 — Prof. Akash Padhiyar

A premium settings screen with persistent dark mode toggle, username storage, and notification preferences.

**Key Features:**
- Dark mode toggle that persists across app restarts
- Username input saved via SharedPreferences
- Personalized greeting using saved name
- Notification toggle with persistent state
- Language selector (visual)
- "Clear All Data" with confirmation dialog
- Smooth theme crossfade transition

**State Management:** `setState` + `shared_preferences`
